# Git คืออะไร

### Git คือระบบสำหรับ เก็บประวัติการเปลี่ยนแปลงของไฟล์ โดยเฉพาะงานเขียนโค้ด

อธิบาย
- รู้ว่าไฟล์ถูกแก้เมื่อไร
- รู้ว่าใครแก้อะไร
- ย้อนกลับไปเวอร์ชันเก่าได้
- แยกงานเป็น branch ได้
- ทำงานร่วมกับคนอื่นได้ผ่าน GitHub / GitLab / Bitbucket

สรุป
- Git = โปรแกรมจัดการ version
- GitHub = เว็บไซต์สำหรับเก็บ repo และทำงานร่วมกัน
